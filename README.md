1. **Установите Node.js** (если не установлен): 
   https://nodejs.org/
2. **Клонируйте репозиторий**
   ```sh
   git clone <URL_REPOSITORY>
   cd <REPOSITORY_FOLDER>
   ```
3. **Установите зависимости**
   ```sh
   npm install
   ```
4. **Запустите сервер**
   ```sh
   npm start
   ```
5. **Откройте в браузере**
   http://localhost:3000
